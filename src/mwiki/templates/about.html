MWiki (v0.1) - Markdown Wiki Application

About

Alternative file-based Wiki that uses a Markdown-based as markup language, based on Github-Flavoured Markdown, MyST markdown from Jupyter Book project, Obisdian Markdown and Mediawiki (Wikipedia's wiki engine) markup language, also known as wikicode.

NOTE: This is a still a work in progress.

Open Source Libraries Used by this project

  • Python Flask Web Framework
  • Markdown_it_py Markdown Parser form Jupyter Book project
  • Python Pygment for server-side syntax highlight of source code blocks.
  • Python Frontmatter library for parsing markdown frontmatter
  • Python Click command line parser
  • Ace JavaScript code editor
  • MathJax JavaScript library for displayin math on the web
